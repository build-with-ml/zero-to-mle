# Contributing to MLE Roadmap

Thanks for your interest in improving this roadmap! Here's how you can help.

## Ways to Contribute

### 1. Fix Errors or Broken Links
- Found a dead link? Submit a PR with the updated URL.
- Found incorrect information? Open an issue with the correction.

### 2. Add Better Resources
- Found a resource that's better than what's listed? Open a PR with:
  - The resource URL
  - Why it's better (more current, more practical, better explained)
  - Which week/phase it belongs to

### 3. Share Your Experience
- Completed a phase? Share what worked and what didn't via a Discussion.
- Got an offer following this roadmap? Share your timeline and adjustments.

### 4. Add Interview Questions
- Add real interview questions you encountered (anonymized, no NDA violations).
- Include which company, role level, and round type.

### 5. Improve Visualizations
- Better diagrams, animations, or visual explainers are always welcome.

## Guidelines

- Keep recommendations **free or low-cost** — accessibility matters.
- All resources must be **current** (last updated within 2 years).
- Prefer **code-first, project-based** resources over theory-only.
- Include the **target company relevance** for any interview-related additions.

## PR Format

```
## What this changes
[Brief description]

## Why
[Why this improvement matters]

## Phase/Week affected
[Which section this modifies]
```

## Code of Conduct

Be respectful, be helpful, be kind. We're all on the same journey.
