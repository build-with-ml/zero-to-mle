## Company-Specific Prep Notes

### Amazon MLE
- 7 rounds: OA → HR → 2 coding → 2 system design → behavioral
- 2 rounds feel like SDE (API design, service architecture)
- LP stories in EVERY round
- SageMaker knowledge is a plus

### Meta MLE
- 5–6 rounds: coding → ML fundamentals → ML system design → behavioral
- "Own the whole ML lifecycle"
- Recommendation systems heavy
- Move fast and ship > perfect theory

### Google MLE
- 5 rounds: coding (×2) → ML system design → system design → behavioral
- Coding matters AS MUCH as ML
- TensorFlow/JAX/TPU knowledge helps
- Design for Google scale (billions of users)

### OpenAI / Anthropic
- PyTorch is mandatory
- Distributed training knowledge (multi-GPU, FSDP)
- RLHF/alignment understanding
- Safety/interpretability focus (especially Anthropic)
- "Research taste" — ability to identify important problems
