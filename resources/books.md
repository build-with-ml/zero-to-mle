# 📖 Recommended Books

## Must-Read (in order)

| # | Book | Author | Phase | Why |
|---|---|---|---|---|
| 1 | Hands-On Machine Learning (3rd ed.) | Aurélien Géron | Phase 2–3 | The definitive practical ML book. Covers sklearn → TensorFlow/Keras. |
| 2 | Deep Learning for Coders with fastai and PyTorch | Jeremy Howard | Phase 3 | Top-down practical approach. Companion to the Fast.ai course. |
| 3 | Designing Machine Learning Systems | Chip Huyen | Phase 5–6 | THE book for ML system design interviews. Production-focused. |
| 4 | Introduction to Machine Learning Interviews | Chip Huyen | Phase 6 | Free. Overview of the ML interview landscape. |

## Supplementary (read relevant chapters)

| Book | Author | Phase | Best For |
|---|---|---|---|
| Mathematics for Machine Learning | Deisenroth, Faisal, Ong | Phase 1 | [Free PDF](https://mml-book.github.io/). Mathematical foundations. |
| Think Stats / Think Bayes | Allen Downey | Phase 1 | Code-first probability. [Free online](https://greenteapress.com/). |
| Pattern Recognition & Machine Learning | Christopher Bishop | Phase 2 | Deep theory. Reference for proofs. |
| Deep Learning | Goodfellow, Bengio, Courville | Phase 3 | The "deep learning bible." Heavy but comprehensive. |
| Natural Language Processing with Transformers | Lewis Tunstall et al. | Phase 4 | HuggingFace-focused NLP. Practical. |
| Build a Large Language Model (From Scratch) | Sebastian Raschka | Phase 4 | Step-by-step LLM implementation. Very 2024/2025. |
| Machine Learning Engineering | Andriy Burkov | Phase 5 | Production ML practices. Short and practical. |
